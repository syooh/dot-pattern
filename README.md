# 🧶 Dot Pattern

사진을 뜨개질 도안으로 변환하고,
웹에서 직접 수정할 수 있는 프로그램입니다.

## ✨ 주요 기능

- 사진 → 뜨개질 도안 변환
- 색상 개수 조절 (12 / 16 / 32 ...)
- JSON 저장
- 빈 도안 생성
- 웹에서 직접 도안 수정 (개발 중)

## 🛠 기술 스택

### Backend
- Python
- Pillow
- K-Means

### Frontend
- React
- TypeScript
- Vite

### 예정

- Django
- MySQL
- PDF 출력
- 사용자 계정
