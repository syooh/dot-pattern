# 개발일지

날짜 : 2026-07-03

## 오늘 목표

프로젝트 구조를 정리하고
Canvas와 PatternEngine을 연결하는 구조를 완성하는 것이 목표였습니다.

## 구현한 기능

- usePattern 리팩터링
- PatternCanvas 리팩터링
- PatternEditor 구조 수정
- ColorPalette UI 개선
- 사용자 색상 추가 기능 개선

## 배운 내용

React에서는 화면을 담당하는 컴포넌트와
상태를 관리하는 Hook을 분리하는 것이 중요하다는 것을 배웠습니다.

또한 PatternEngine을 따로 만들어
도안 수정 로직을 분리하면
Undo, Redo, 브러시 등의 기능을 쉽게 추가할 수 있다는 것을 알게 되었습니다.

## 느낀 점

오늘은 새로운 기능을 많이 추가하기보다
프로젝트의 기반을 다지는 작업을 진행했습니다.

지금 구조를 잘 만들어 두면
나중에 기능을 추가할 때 훨씬 수월해질 것입니다.

## 내일 목표

- 색상 삭제 기능 구현
- Undo / Redo 기능 설계
- PatternEngine 기능 확장
