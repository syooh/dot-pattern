# 개발일지

날짜 : 2026-07-03

## 오늘 구현한 기능

- PatternCanvas 리팩터링
- ColorPalette 리팩터링
- 사용자 색상 추가 기능 구현

## 배운 내용

Canvas는 화면만 담당하고
실제 데이터 수정은 PatternEngine에서 처리하도록
역할을 분리하였다.

이 구조를 사용하면
Undo, Redo 등의 기능을 쉽게 추가할 수 있다.

## 느낀 점

컴포넌트의 역할을 분리하는 것이
유지보수에 매우 중요하다는 것을 배웠다.
