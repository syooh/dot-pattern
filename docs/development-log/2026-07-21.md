# ✅ 작업 내용

## 1. Zoom 좌표 보정

### 문제

Canvas를 확대하면 마우스가 위치한 Cell과 실제 색칠되는 Cell이 서로 달랐다.

### 원인

마우스 좌표를 Cell 좌표로 변환하는 과정에서 Zoom 값이 적용되지 않았다.

### 수정

`pixelToCell()` 함수가 `camera.zoom`을 사용하도록 수정하였다.

```text
Mouse Position
        │
        ▼
pixelToCell(pixel, zoom)
        │
        ▼
정확한 Cell 계산
```

### 결과

* 확대 상태에서도 정확한 위치에 Paint 가능
* Hover 위치와 Paint 위치 일치

## 5. Delete 기능 구현

Selection 내부 Pixel을 삭제하는 기능을 구현하였다.

### PatternEngine

범용 함수 추가

```
fillSelection(pattern, selection, colorId)
```

Delete는

```
colorId = 0
```

으로 호출하여 흰색으로 변경하도록 구현하였다.

### 처리 흐름

```text
Delete

↓

Selection 확인

↓

fillSelection()

↓

clonePattern()

↓

선택 영역 반복

↓

pixels[y][x] = 0

↓

setPattern()

↓

Canvas Render
```

## 6. TypeScript 타입 안정성 개선

Delete 구현 과정에서

```
PatternData | null
```

타입 오류가 발생하였다.

### 수정

Pattern과 Selection이 존재하는지 확인 후 처리하도록 변경하였다.

```ts
if (!pattern || !selection) {

    return;

}
```

불필요한 Non-null Assertion(`!`)을 사용하지 않고 안전하게 처리하도록 수정하였다.
