# 📝 Result

Canvas 엔진의 구조가 대부분 완성되었다.

Render 구조가 Layer 기반으로 변경되면서 앞으로 Selection, Zoom, Guide Line 등의 기능을 기존 코드를 크게 수정하지 않고도 추가할 수 있는 기반이 마련되었다.
